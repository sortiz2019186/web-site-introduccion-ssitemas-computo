TEMAS DEL CURSO

Cada uno de los temas a continuación se ha visto a la largo de estas 11 semanas del primer semestre de Ingeniería en Sistemas de la Información y ciencias de la computación. A continuación, se da una vista previa de cada uno de los temas vistos.

INTRODUCCIÓN A LAS COMPUTADORAS

Una computadora es un sistema compuesto por diferentes elementos físicos llamados HARDWARE que se relacionan a través de instrucciones por medio del SOWFTWARE.

Leer más

SISTEMAS OPERATIVOS

Un sistema operativo (SO) es un software que actúa como intermediario entre el hardware de un dispositivo de computación y las aplicaciones de software. Es esencialmente el sistema de gestión principal que coordina y controla todas las actividades y recursos del sistema.

Leer más

BASE DE DATOS

Una base de datos es un conjunto organizado de datos relacionados entre sí y estructurados de manera que se puedan acceder, administrar y actualizar de forma eficiente. En esencia, una base de datos es un sistema que permite almacenar, recuperar, modificar y eliminar información de manera estructurada y segura.

Leer más

REDES DE COMPUTADORAS

Las redes de computadoras son sistemas interconectados que permiten la comunicación y el intercambio de información entre dispositivos informáticos, como computadoras, servidores, dispositivos móviles, impresoras, entre otros. Estas redes pueden ser cableadas o inalámbricas y se utilizan para compartir recursos, datos y servicios entre los dispositivos conectados.
